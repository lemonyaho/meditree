04 분노의 대처
04.1 분노 다루기
→ 분노를 인정
→ 분노의 원인을 분별
→ 생각을 자제
→ 모든 일을 간섭하지 않는다
→ 불합리한 신념에 도전
→ 대인관계의 목적을 생각
→ 마음의 평화를 발전
04.2 적절히 분노한다
→ 사적인 자리에서 표현
→ 상대방의 잘못이 아닌 감정의 관점에서 말한다
→ 논쟁의 초점을 문제의 핵심에 집중한다
→ 논쟁을 이기겠다는 태도를 포기
→  부드럽고 긍정적인 분위기
→ 문제를 과장하거나 관계단절의 위협은 하지 않는다
→ 반응을 보일 기회를 허락하고 중간에 방해하지 않는다
→ 상대방에 주의집중한다
→ 상대방이 한 말을 풀어서 설명
→ 서로 정확히 알아들었는지 확인
→ 해결을 향한 방향으로 나가는 것에 집중
04.3 분노의 예방
→ 대처할 준비가 되지 않은 상황을 피하기
→ 과중한 감정적 부담을 피하기
→ 건강과 휴식을 소홀히 하지 않기
→ 자존감을 지키기
→ 혀를 조심하기
→ 정직한 대화를 개발하기
→ 묵은 분노를 제거하기
→ 현재의 관계에 집중하기
→ 경쟁을 피하기
04.4. 환자가 화를 내는 경우
→ 자신부터 화를 참는다
→ 실제 이유는 다른 사람, 다른 일에 화를 내고 있다고 명심
→ 화를 내는 진짜 이유를 찾기
→ 화가 난 감정을 인식시키기
